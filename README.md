# MarketPlay

An interactive financial learning game built with Next.js.

Teaches how money, markets, and decisions work through real-time scenarios, instead of boring lessons.

Users make decisions, watch outcomes unfold, and build real financial intuition in a fast, game-like experience.

## Live Demo
https://market-play-lac.vercel.app

## How to run locally
1. Clone the repo
2. `npm install`
3. `npm run dev`
4. Open http://localhost:3000

Built with ❤️ using Next.js, React, and custom styles.
